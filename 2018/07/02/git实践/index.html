  2. 如何将撤销文件的修改

撤销文件的修改其实有很多方法，可以使用编辑器的撤销快捷键。但当你关闭编辑器后，你会发现你用不了撤销了。还有一种情况是你修改了n个文件时，你想批量撤销修改时，你发现如果是快捷键的方式是多么的繁锁。这时最后的方式就是使用git命令不解决了。

    1. 使用命令git status， 查询工作区有哪些修改过的文件，如图

    2. 再使用命令git checkout -- <file>，就可以撤销指定文件的修改，如图

    3. 如果要撤销所有，使用命令git checkout .。
